# Infinite Platformer Game in Pygame

## Description

This is a simple infinite platformer game built using **Python** and **Pygame**. The player controls a character that can move left, right, and jump onto continuously generated platforms. The game is designed to be infinite and undefeatable—the player cannot lose.

## Features

- **Infinite Gameplay:** Platforms are automatically generated as you progress.
- **Simple Controls:** Move left, right, and jump using the keyboard.
- **Basic Physics:** Includes gravity and collision detection.
- **No External Assets:** Uses simple shapes; no images or sounds required.

## Getting Started

### Prerequisites

- **Python 3.x**
- **Pygame**

### Installation

1. **Clone the Repository**

   ```
   git clone https://github.com/spsokhi/PlatformerGame-Py.git
   cd PlatformerGame-Py
2. **Create a Virtual Environment**
   On Windows:
   ```
   python -m venv venv
   ```
   On macOS/Linux:
   ```
   python3 -m venv venv
3. **Activate the Virtual Environment**
   On Windows:
   ```
   venv\Scripts\activate
   ```
   On macOS/Linux:
   ```
   source venv/bin/activate
4. **Install Dependencies**
     ```
     pip install pygame

## Running the Game
With the virtual environment activated, run:
   ```
    python main.py
   ```
## How to Play
- Move Left: Press the Left Arrow key.
- Move Right: Press the Right Arrow key.
- Jump: Press the Space Bar.

## Objective
+ Navigate your character upward by jumping on platforms.
+ The game is infinite, so enjoy playing without the worry of losing!
